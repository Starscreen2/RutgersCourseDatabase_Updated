Rutgers Course API

Real-time course information API with automatic updates

Endpoints

GET /api/courses

Retrieve course information with optional filters

Query Parameters

  • subject (optional) - Filter by subject code
  • course_number (optional) - Filter by course number

Example Request

curl http://localhost:5000/api/courses?subject=CS&course_number=111

Example Response


{
    "status": "success",
    "data": [...],
    "last_update": "2024-02-15T10:30:00"
}

GET /api/health

Check API health status

Example Response


{
    "status": "healthy",
    "last_update": "2024-02-15T10:30:00"
}

Rate Limits

  • 200 requests per day
  • 50 requests per hour
  • 30 requests per minute for /api/courses
