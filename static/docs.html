Endpoints

GET /api/courses

Retrieve course information with optional filters

Query Parameters

  • subject - Filter by subject code (e.g., CS)
  • course_number - Filter by course number (e.g., 111)

Example Request

curl http://localhost:5000/api/courses?subject=CS&course_number=111

Example Response


{
    "status": "success",
    "data": [
        {
            "courseString": "01:198:111",
            "title": "Introduction to Computer Science",
            "subject": "CS",
            "courseNumber": "111",
            "credits": "4",
            "sections": [...]
        }
    ],
    "last_update": "2024-01-15T10:30:00"
}

GET /api/health

Check API health status

Example Request

curl http://localhost:5000/api/health

Example Response

{
    "status": "healthy",
    "last_update": "2024-01-15T10:30:00"
}

Rate Limits

  • 200 requests per day
  • 50 requests per hour
  • 30 requests per minute for /api/courses endpoint
